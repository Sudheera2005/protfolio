# Sudheera Perera | Portfolio Website

Welcome to my personal portfolio website. This project showcases my work, skills, and background in full-stack development, software engineering, and cybersecurity.

## 🚀 Live Demo
Visit the website at: **[sudheera2005.github.io/portfolio](https://sudheera2005.github.io/portfolio)**

---

## 🎨 Design & Aesthetic System
The project is built around a **"Software Architecture meets Cybersecurity"** visual theme:
- **Core Aesthetic**: Premium dark mode featuring neon red and hot pink accents, frosted-glass components (glassmorphism), and subtle code/terminal bracket styling (`> ` prompts, `[ ]` frames).
- **Interactions**: Interactive background canvas with moving nodes/particles, smooth slide-in animations on scroll (AOS), and micro-animations on interactive badges/cards.
- **Section Layout**:
  - **Hero**: Clean welcome greeting, dynamic typing animation, floating ring-glowing profile, and custom action links.
  - **About Me**: A redesigned overview containing Client/Project Focus areas, technical details, a "Current Focus" skill-level tracker, and direct quote cards.
  - **Skills & Technologies**: Auto-scrolling marquee carousel followed by detailed, categorized skills grids (Languages & Frameworks, Security & Networking, Databases & Architecture, Tools & Practices) and a visually distinct Leadership card featuring text context.
  - **Projects Grid**: Highlighted works (logistics systems, Palettes Stress-Test Engine, client e-commerce platforms, network designs) styled with interactive overlays.
  - **Contact Panel**: Clean feedback form and links to professional profiles (GitHub, LinkedIn).

---

## 🛠️ Technology Stack
- **Structure**: HTML5 (Semantic, SEO optimized)
- **Styling**: Vanilla CSS3 (Custom properties/variables, media queries for 100% responsiveness)
- **Interactions**: Vanilla JavaScript (typing effect, navigation controls, particle canvas)
- **Animations**: AOS (Animate on Scroll)
- **Icons**: Boxicons (Outline-style icons matching site weights)
